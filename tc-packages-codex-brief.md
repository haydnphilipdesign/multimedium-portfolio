# TC Website Packages Pricing Strategy — Codex Implementation Brief

This file summarizes the recommended pricing and positioning changes for MULTIMEDIUM's Transaction Coordinator website packages. Use this as context for updating the TC packages page, package cards, pricing tables, FAQ copy, referral positioning, and any related package data files.

---

## Strategic Direction

The current MULTIMEDIUM positioning is strong. The niche focus on transaction coordinators, the live demos, and the founder's connection to the TC world all support credibility. The main issue is not design quality. The issue is the low-end pricing ladder and the way the value is explained against DIY options like Wix, Squarespace, GoDaddy, or low-cost Fiverr designers.

The package structure should be expanded from three tiers to four tiers so beginner TCs have a clear on-ramp without forcing MULTIMEDIUM to discount custom work.

The new lowest tier should be tightly constrained and positioned as a streamlined, template-based presence. It should not become a custom design project at a lower price.

---

## Final Recommended Package Structure

### 1. TC Presence — $595

**Positioning:**  
A professional one-page credibility site built on a proven layout. This is not a fully custom website. It is a streamlined, customized web presence for new solo TCs who need to look legitimate quickly.

**Best for:**  
New solo transaction coordinators with little or no deal flow who need a professional web presence when an agent searches for them.

**Key inclusions:**

- One-page website
- Built from a proven layout or small set of layout options
- Client branding applied: logo, colors, photos, basic business details
- Guided copy based on client answers
- Mobile-responsive design
- Basic SEO/meta setup
- Open Graph/social link preview setup
- Clear contact CTA
- Fast turnaround

**Hard scope boundaries:**

- No intake form embedding
- No separate submit page
- No advanced form logic
- No custom section ordering beyond the approved layout structure
- No full copywriting from scratch
- No deep strategy/discovery process
- One draft and one light refinement pass only

**Goal:**  
This package gives price-sensitive but serious beginners a professional entry point without cheapening the main custom packages.

---

### 2. TC Starter — $795

**Positioning:**  
A one-page TC website with a basic intake/contact workflow so agents have a clear way to start working with the TC.

**Best for:**  
Solo TCs who are beginning to work with agents and want more than just an online business card.

**Key inclusions:**

- Everything in TC Presence
- Embedded intake/contact form section or simple submit workflow
- More complete service explanation
- Stronger CTA flow for agents
- Basic file-start instructions
- One refinement round
- Launch support

**Important distinction from TC Presence:**  
TC Presence is for credibility only. TC Starter is for credibility plus a simple agent action flow.

**Pricing instruction:**  
Use **$795** as the public price for TC Starter. Do not use $750 or $895. This keeps the tier approachable while creating a cleaner step up from the $595 TC Presence package.

**Goal:**  
Make $795 feel meaningfully different from the $595 tier by connecting it to business workflow, not just page design.

---

### 3. TC Growth — $1,495

**Important pricing instruction:**  
TC Growth should be priced at **$1,495**, not $1,500 and not $1,249.

**Positioning:**  
A polished multi-page business website with a dedicated submit/intake flow, clearer service positioning, and a stronger professional presentation.

**Best for:**  
Established TCs who want stronger branding, better service communication, and a real intake workflow.

**Key inclusions:**

- Multi-page website
- Recommended structure: Home, Services, Submit a File, About or Careers
- Dedicated submit/intake page
- Embedded intake form(s), such as Jotform, Cognito, Tally, Typeform, etc.
- Strategic service positioning
- Collaborative copywriting
- Mobile-responsive design
- SEO/meta setup
- Open Graph/social link preview setup
- Polished interactions and stronger visual presentation
- Launch support

**Pricing rationale:**  
$1,495 preserves the premium positioning of the Growth tier while making the price feel slightly more approachable than a hard $1,500. Do not lower this tier to $1,249. Lowering it too much would weaken the perceived value and compress the pricing ladder.

**Goal:**  
This should remain the primary recommendation for serious TCs and referral partners. It is the best balance of value, margin, and deliverability.

---

### 4. TC Pro — $2,495+

**Positioning:**  
A custom website for TC teams, coaches, referral-driven operators, or higher-volume businesses that need stronger positioning, deeper trust-building, and a more complete lead/conversion system.

**Best for:**

- TC teams
- TC coaches or educators
- High-volume operators
- Businesses with referral partners
- TCs who need testimonials, advanced intake, additional pages, or deeper brand strategy

**Key inclusions:**

- Everything in TC Growth
- Additional pages as needed
- Fully written or more heavily supported copy
- Deep discovery/positioning process
- Testimonials/social proof strategy
- Referral partner landing page options
- Advanced intake options, including multi-step or conditional forms when scoped
- Stronger conversion strategy
- Analytics setup if appropriate
- More robust post-launch support

**Pricing instruction:**  
Use **$2,495+** as the public starting price. This keeps the tier premium while reducing the psychological wall of $3,000+ for small service businesses.

## Messaging Guidance

The page should avoid framing the packages only around page count. Page count matters, but it should not be the main perceived value.

The copy should focus on what the website helps the TC accomplish:

- Look legitimate when agents search for them
- Explain services clearly
- Reduce back-and-forth questions
- Give agents a clear next step
- Create a professional submit-a-file/intake process
- Make the TC feel more established than a Facebook page, Canva flyer, or half-built DIY site

---

## DIY Comparison Positioning

Add copy that directly addresses why this is not the same as using Wix, Squarespace, GoDaddy, or Fiverr.

Suggested copy direction:

> This is not a generic template you have to figure out yourself between closings. These packages are built around the way transaction coordinators actually get found, evaluated, and contacted by agents. You answer guided questions about your business. I handle the structure, copy flow, design, mobile experience, and launch details so your site feels professional from day one.

Potential FAQ:

### Can I just build this myself on Wix or Squarespace?

Yes, you can. But many new TCs start there and end up with a half-finished site, generic copy, and no clear workflow for agents to submit a file. These packages are for TCs who want the site handled properly so they can focus on getting clients and closing files.
